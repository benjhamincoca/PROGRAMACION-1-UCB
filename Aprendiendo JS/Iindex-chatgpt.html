Guía rápida

  • Coloca tus funciones en aprendiendo.js y exportalas a window si usas módulos.
  • La plantilla incluye llamarMiAlgoritmo() que intenta invocar mi_primer_algoritmo() y muestra su resultado en el panel Salida.
  • Diseño responsive y accesible listo para producción.

Sobre aprendiendo.js

Crea el archivo aprendiendo.js en el mismo directorio y define function mi_primer_algoritmo() { ... } o asigna window.mi_primer_algoritmo = () => { ... } si trabajas con módulos.

Buenas prácticas

  1. Devuelve resultados — evita alertas; usa return y deja que la interfaz los muestre.
  2. Documenta tus funciones con comentarios o JSDoc.
  3. Mantén lógica separada de UI: cálculos en aprendiendo.js, render en HTML.
